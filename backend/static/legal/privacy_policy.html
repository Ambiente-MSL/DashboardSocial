MSL Monitor · Política de Privacidade

Como cuidamos dos seus dados

Documento público para o Facebook App Review e para todos os usuários do MSL Monitor.

1. Quem somos e escopo

O MSL Monitor é uma solução da MSL Estratégia voltada para times internos analisarem métricas do Facebook e Instagram. Este documento explica como coletamos, usamos e protegemos dados pessoais e de contas conectadas.

Contato do controlador: privacidade@mslestrategia.com.br

2. Dados que coletamos

  • Credenciais da conta interna: nome, e-mail corporativo e senha (armazenada com hash PBKDF2). Apenas usuários autorizados acessam o painel.
  • Dados da Meta (após consentimento no login do Facebook): ID da Página/Conta de Anúncios, ID do Instagram Business, métricas agregadas (alcance, engajamento, seguidores) e posts públicos necessários para gerar relatórios.
  • Dados de uso: registros de acesso, data/hora e parâmetros das requisições para segurança e suporte.

Não coletamos dados sensíveis nem conteúdos privados. Tokens de acesso são usados apenas para leitura de métricas autorizadas.

3. Como usamos os dados

  • Autenticar usuários e permitir acesso seguro ao dashboard.
  • Gerar indicadores, gráficos e relatórios de desempenho para as contas conectadas.
  • Monitorar estabilidade, prevenir abuso e atender solicitações de suporte.
  • Cumprir requisitos do Facebook App Review, incluindo exibir o fluxo de autenticação.

Não vendemos, alugamos ou usamos dados para publicidade de terceiros.

4. Base legal e modelo B2B

Processamos dados com base em legítimo interesse e execução do contrato para atender equipes autorizadas do controlador. O login com Facebook é opcional e serve para que revisores e gestores comprovem a autorização de acesso às métricas.

5. Compartilhamento e terceiros

  • Infraestrutura: provedores de cloud, banco de dados e monitoramento usados para hospedar o MSL Monitor.
  • Facebook/Instagram: apenas para ler dados autorizados via Graph API.

Não compartilhamos dados com parceiros comerciais ou anunciantes.

6. Retenção e exclusão

  • Dados de conta e logs de acesso são mantidos enquanto a organização usar o serviço ou até pedido de exclusão.
  • Métricas históricas são armazenadas para comparativos; podemos removê-las mediante solicitação do controlador.
  • Tokens e sessões expiram periodicamente e podem ser revogados a qualquer momento pelo usuário na Meta.

7. Segurança

  • Tráfego protegido por HTTPS.
  • Senhas armazenadas com hash forte (PBKDF2) e tokens guardados de forma segura.
  • Controles de acesso por função (analista/admin) e logs para auditoria.

8. Seus direitos

  • Acessar, corrigir ou solicitar exclusão dos dados de conta.
  • Revogar o acesso do aplicativo no painel do Facebook/Instagram a qualquer momento.
  • Solicitar esclarecimentos pelo e-mail privacidade@mslestrategia.com.br.

9. Alterações

Podemos atualizar este aviso para refletir mudanças no produto ou em requisitos regulatórios. A versão mais recente estará sempre disponível neste endereço.
